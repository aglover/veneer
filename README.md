This project front's the App47 API (which requires a custom HTTP header; thus, making it hard to use in simple REST client libraries). This app is intended to be deployed to Heroku and is accordingly associated with the [veneer](http://veneer.herokuapp.com/) heroku app. 

For more information on App47's API, see:

* [Obtaining App data programmatically](http://www.app47.com/2011/09/21/obtaining-app-data-programmatically/)
* [api-examples Github project](https://github.com/App47/api-examples)
